# Autonomous chrome dino game player 

A real-time program that plays the Chrome Dino game
by detecting obstacles from screen captures and automatically jumping/ducking.

## Tech Stack

- Python
- MSS (screen capture)
- PyAutoGUI (input automation)
- OpenCV
- ... (detta är preliminärt)

## How It Works

1. Capture a region of the screen (game area)
2. Process the image using tools like OpenCV
3. Detect obstacles and dino speed
4. Trigger keyboard input to jump/duck at the optimal time
